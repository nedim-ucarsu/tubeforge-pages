TubeForge Privacy Policy

Last updated: July 22, 2026

TubeForge is an individually developed desktop application operated by Nedim Uçarsu. Questions or deletion requests may be sent to upwamzn@gmail.com.

Data processed

After a user grants Google OAuth authorization, TubeForge may process authorization tokens, channel and video identifiers, video metadata, captions, thumbnail and upload status, and YouTube Analytics metrics. TubeForge never asks for or stores a user's Google password.

Purpose and use

Google user data is used only to provide user-requested functions on authorized YouTube channels: channel inventory, private-first video upload, metadata updates, caption and thumbnail management, and performance analysis. TubeForge does not use Google user data for advertising or artificial engagement.

Storage, sharing and retention

Application data and authorization credentials are stored in the operator-controlled desktop environment and protected with reasonable technical safeguards. Data is not sold. It is shared only with Google/YouTube API Services as necessary to perform requested operations. Data is retained only while needed for the authorized workflow and can be deleted upon request or when authorization is removed.

User control

Users can revoke TubeForge's Google access at any time from Google Account permissions. They may also request deletion of locally retained application data by contacting the address above.

Compliance

TubeForge's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements, and to the YouTube API Services Terms of Service.
