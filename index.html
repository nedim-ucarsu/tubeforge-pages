TubeForge Desktop

TubeForge is an individually developed desktop application for producing and managing multilingual YouTube content on channels authorized by the user.

Purpose

The application combines source-reviewed scripts, user-provided or licensed visual assets, narration, subtitles, and metadata into localized video packages. It is currently a private development and testing tool and is not publicly downloadable.

YouTube API use

TubeForge uses YouTube API Services only for channels explicitly authorized through Google OAuth. Its functions include reading channel and video inventory, uploading videos as private by default, updating video metadata, uploading captions and thumbnails, and reading YouTube Analytics metrics for evaluation.

TubeForge does not request Google passwords, access unauthorized channels, sell user data, automate views, likes, comments or subscriptions, or generate artificial engagement.

Development program

The present program is a six-month multilingual content and measurement study across ten authorized test channels. Each channel remains isolated and all publishing decisions stay under the operator's control.
